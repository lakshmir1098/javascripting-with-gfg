1. Using `+` Operator
    ```js
    let sum = num1 + num2;
    ```

2. Using function
    ```js
    function additionFunction(a, b) {
        return a + b;
    }
    ```
3. Using Arrow function
    ```js
    let addition = (a, b) => a + b;
    ```
4. Using Addition Assignment (+=) Operator
    ```js
    let num1 = 15;
    let num2 = 10;
    num1 += num2;
    ```
