1. Using `*` Operator
    ```js
    let result = a * b
    ```
2. Using Functions
    ```js
    function fun_name (param1 , param2) {
        return param1 * param2 ;
    }
    ```
    OR

    ```js
    function fun_name (param1 , param2){
        return param2 * param1 ;
    }
    ```
3. Using Arrow function
    ```js
    let result = (a, b) => a * b;
    ```
4. Using Multiplication assignment operator
    ```js 
    let a = 20; 
    a *= 10; 
    let b = 2; 
    a *= b; 
    ```
4. Using for loop
    ```js
    function multiply(a, b) { 
        let result = 0; 
        for (let i = 1; i <= b; i++) { 
            result += a; 
        } 
        return result; 
    } 
    ```
